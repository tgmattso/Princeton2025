# Science And Computing Lectures

This repository holds materials for my Wintersession 2025 lectures.  This includes the following.

* An Introduction to Parallel Programming using OpenMP

* Floating Point Numbers are not Real

* Ai and the future of programming

While the Parallel Programming lecture is driven by hands-on exercises, the other two (Floating Point and AI) are pure lectures.

These materials will be online for about a month following the lectures.  After that, this repository will be deleted.  For archival storage of this content, see my more ``permanent'' repositories:

* ParProgForPhys - this is the repository where I keep my hands on course on parallel computing.

* SciCompHPC - this is lecture and exercises for a 10 week course in scientific computing.  It has both hands-on and lecture content.

The above repositories are under continuous refinement.  I have no plan to ever delete them.
